Welcome on the FeynmanGraph Javascript Library Webpage (FGJL)

What is FGJL ?

The FGJL is a powerful tool to create fancy and animated Feynman Diagrams

But more than just allow the user to draw some line and point the FGJL embedded a physics rules checker to be sure that the user creation is right and an high level creation assistant !

After the step of the creation it's possible to export your draw under some general formats like png,pdf and others...

It's possible to get a code to add to your website code to print your animated feynman diagram. The creation is book markable too.

In a more complex way, the physics engine behind is able to compute a lots of interesting value like the relative cross section which give to this library a real interest no only for beginners but also for physicists.

Origin

The FGJL is at the origin a part of the Standard Model Project. This project is born during the CERN Webfest which is an incredible weekend of creativity, friendship and learning.
